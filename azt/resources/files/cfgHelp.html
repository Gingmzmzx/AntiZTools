软件配置帮助页面

本页面提供对软件配置文件各项设置的解释和指导。

1. 更新设置

字段 类型 默认值 说明
version 整数 114 软件的当前版本号。
forceUpdate 布尔 True 当配置版本过低时，是否强制更新配置为新版的默认配置。

2. 配置设置

字段 类型 默认值 说明
StartShow 布尔 False 启动时是否显示主界面。
EXEFileName 字符串 AntiZTools.exe 主程序的执行文件名。
Password 字符串 ZBZ666 访问软件的密码。
RegAppName 字符串 AntiZTools 注册的应用程序名称。（通常用于开机自启注册表）
CheckUrl 字符串 https://example.com 检查网络连接的URL地址。
DESKey 字符串 cc757e12a8d15d65 DES的密钥。

3. 字体设置

字段 类型 默认值 说明
Enable 布尔 False 是否启用自定义字体。
FontFamily 字符串 幼圆 字体名称。
FontPointSize 整数 10 字体大小。

4. 插件设置

字段 类型 默认值 说明
Enable 布尔 True 是否启用插件功能。
Path 字符串 plugins 插件文件的路径。（~/.AntiZTools/{Path}）
Disable 数组 [] 禁用的插件列表。

5. 新闻设置

字段 类型 默认值 说明
NewsApi 字符串 https://azt.xzynb.top/news/news.json 新闻源的 API 地址。
Enable 布尔 False 是否启用新闻功能。

6. 临时文件清理

字段 类型 默认值 说明
CleanFailedTimes 整数 2 当失败文件数目大于此值时，会提示。
TempFilePrefix 字符串 _MEI 临时文件的前缀。

7. 通知设置

字段 类型 默认值 说明
Messager 对象 { "MaxWidth": 300, "MinWidth": 300 } 通知消息的宽度设置。
NotifyWindow 对象 { "Height": 300, "Width": 200 } 通知窗口的高度和宽度。（新闻资讯弹窗属于NotifyWindow）

8. 窗口标题设置

字段 类型 默认值 说明
Enable 布尔 True 是否启用窗口标题功能。
Interval 整数 1 处理间隔时间。
hook 数组 [] 窗口标题处理的具体规则，包括：
  • title: 窗口的标题。
  • enable: 是否启用该规则。
  • fuzzyMatching: 是否启用模糊匹配。
  • handler: 处理该窗口的操作。
  • data: 处理数据，如消息内容和通知设置。

9. 托盘设置

字段 类型 默认值 说明
ToolTip 字符串 GeoGebra 托盘图标的提示信息。
StartMsg 字符串 已在后台运行 启动时的消息提示。

10. 屏幕保护程序设置

字段 类型 默认值 说明
StartScreenSaverTimer 整数 0 屏幕保护程序启动的定时器设置。
StartScreenSaverLastTime 整数 1800 屏幕保护程序的持续时间（秒）。
ScreenSaverCmd 字符串 D:\HiteVision\electron.exe D:\HiteVision 启动屏幕保护程序的命令。
